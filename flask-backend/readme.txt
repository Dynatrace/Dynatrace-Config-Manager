Setup:
	Only tested in wsl2 Ubuntu
	Python version: 3.9
	Install pipenv
		installing pipenv can be clunky, but it's worth it
		Don't forget to update your PATH with:
			PATH="$PATH:$HOME/.local/bin

	Go to the root of the flask backend and run:
		pipenv install

Run the Tool:
	sh run_flask.sh

Note:
	This is only the backend.
